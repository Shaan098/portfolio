🎯 CUSTOMIZATION GUIDE

Everything in your portfolio is EASY to customize:

Change Your Name:
   → Edit: frontend/src/config/portfolio.config.js
      personal.name = "Your Name"

Update Your Bio:
   → Edit: frontend/src/components/About.jsx
      Change the bio text

Add Your Projects:
   → Edit: frontend/src/components/Projects.jsx
      Add project objects to the array

Update Skills:
   → Edit: frontend/src/config/portfolio.config.js
      Update skill arrays

Change Colors:
   → Edit: frontend/tailwind.config.js
      Modify color values

Update Contact Info:
   → Edit: frontend/src/components/Contact.jsx
      Update email, phone, links
